Fr. 7b. Heb treats of genders. Varro says: “Only those are genera ‘genders’ which generant ‘generate’; those are properly called genera.” But if we follow his authority, there will be only two genders, masculine and feminine. For no gendersc can procreate except these two.

Fr. 8. If ostrea ‘oyster’ is of the first declension, like Musa ‘Muse,’ it will be declined in the feminine gender, so that we refer the word to the living being; if we use it for the shell, then the word must be ostreum, inflected in the neuter and according to the second declension, so that it is genitive ostrei, dative ostreoa: because Varro says: “No living creature has a name which is inflected in the neuter gender.”b

Fr. 9.
